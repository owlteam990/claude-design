  Design → Code 交接            → 已有 Claude Code 流程，天然闭环

  推荐实现方案（MVP）                                             
                                                                                                                                                      
  3 个阶段：
                                                                                                                                                      
  1. Phase 1 — 对话生成 + 预览（2-3天）                                                                                                               
    - 在桌面端新增 /design 标签页
    - 对话输入 → Claude 输出 HTML/CSS artifact → Tauri WebView 实时渲染                                                                               
    - 支持"重新生成"/"微调"对话                                                                                                                       
  2. Phase 2 — 品牌系统 + 内联编辑（3-4天）                                                                                                           
    - 读取项目代码库提取颜色/字体/组件                                                                                                                
    - 右侧控制面板：颜色选择器、间距滑块、字体切换                                                                                                    
    - 修改实时反映到预览                                                                                                                              
  3. Phase 3 — 导出（1-2天）                                                                                                                          
    - 导出为 HTML 文件 / PDF（Tauri print API）/ PPTX（pptxgenjs）                                                                                    
    - 生成可分享的本地 URL（通过现有 bridge/remote 能力）                                                                                             
                                                                                                                                                      
  ---                                                                                                                                                 
  结论：完全可行，且这个项目是目前最适合做这件事的基础之一。 最大优势是桌面 WebView 可以直接渲染 Claude 生成的 HTML，不需要额外的渲染服务。
